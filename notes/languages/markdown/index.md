# Markdown

> 2026-6-28

## Markdown是什么？

[Markdown](https://markdown.com.cn/basic-syntax/index.html)是一种轻量级标记语言，旨在通过简单的语法快速生成格式化文档。它支持将纯文本转换为HTML、PDF等多种格式，广泛应用于博客、笔记和技术文档的编写。

## Markdown的基本语法

- 标题：标题通过#表示，#的数量决定标题的级别。

```
# 一级标题
## 二级标题
### 三级标题
```

- 字体样式：把要修改的字体用 * 或 _ 符号包裹起来。

```
*斜体* 或 _斜体_
**加粗** 或 __加粗__
***斜体加粗*** 或 ___斜体加粗___
```

- 引用：在句首加 > 。

```
> 这是一个引用
```

- 换行：换行可以通过在行尾添加两个 空格 或插入 br 标签。

- 链接和图片：通过 方括号和圆括号 的格式插入链接或图片。

```
[链接名称](链接地址)
![图片描述](图片地址)
```

- 列表：无序列表在句首加上 - ，有序列表在句首加上 数字 和 . 。

```
- 无序列表项
1. 有序列表项
```

- 分割线：可以通过三个或以上的 - 或 * 实现。

```
---
***
```

- 代码块：使用`包裹一行代码，使用```包裹多行代码。

```
    `单行代码`
    
    ```
    代码片段
    ```
```

- 转义字符：要显示原文档的字符，请在字符前面添加反斜杠字符 \。

```
\ 转义文本
```
